Comandos de Github

Descripción Comandos
Clonar un repositorio git clone [url del repoaitorio que se quiere clonar]
Crear un nuevo repositorio local git init
Agregar los cambios al repositorio git add . o el nombre del archivo que queramos agregar
Crear un commit git commit -m "descripción del cambio realizado"
Hacer el push git push origin main/master"
Sincronizar el repositorio local con el repositorio remoto git remote add origin [url del repositorio]
Crear una rama git branch [Nombre de la rama] o git checkout -b [Nombre de la rama]"
Borrar una rama git branch -d [Nombre de la rama]"
